        AssignWatershedID_archydro(Geodatabase, WatershedName, {Overwrite_WatershedID})
        
           For a given workspace, update the value of WatershedID field of all the
           layer/tables in the workspace. If a layer does not contain the WatershedID
           field, the field will be added.  If a WatershedName is provided instead of a
           WatershedID, the code would find the WatershedID based on the specified
           WatershedName  The tool works with the ApDataExchangeConfig.xml that provide
           (1) a list of valid WatershedName, and WatershedIDs  (2) a list of
           layernames/tablenames not participate in the operation  (3) a list of
           layernames/tablenames that will participate in the operation  <SPAN /> <SPAN />

             WatershedName whose corresponding WatershedID will be used to update the
